# 沫欢记录 (MoHuan Record) - 专属空间 App

一个专为2人设计的 Android 应用程序，提供聊天、视频分享、共同相册、想念传话等甜蜜功能，记录你们的点点滴滴。

## ✨ 功能特点

*   **视频分享**: 上传并分享你们的生活视频，支持点赞、评论和弹幕。
*   **情侣空间**: 绑定另一半，开启专属二人世界。
*   **共同相册**: 上传照片和视频到情侣相册，共同维护美好回忆。
*   **想念传话 (Miss You)**: 发送全屏心跳、极光、烟花等特效，让对方在打开 App 时感受到你的思念。
*   **私密聊天**: 实时聊天功能，支持撤回、状态显示（在线/离线）。
*   **帖子社区**: 发布文章、图片或音乐，记录生活心情。
*   **观看历史**: 记录你的观看足迹。
*   **一起看**: (Beta) 实时同步观看视频功能。

## 🛠 技术栈

### 前端 (Android App)
*   **语言**: Kotlin
*   **UI 框架**: Jetpack Compose (Material Design 3)
*   **架构**: MVVM + Hilt (依赖注入)
*   **网络**: Retrofit + OkHttp
*   **图片加载**: Coil
*   **视频播放**: ExoPlayer
*   **导航**: Navigation Compose

### 后端 (Node.js)
*   **运行时**: Node.js
*   **Web 框架**: Express
*   **数据库**: JSON 文件存储 (轻量级，无需安装数据库软件)
*   **身份验证**: JWT (JSON Web Tokens)

## 📂 目录结构

*   `app/`: Android 客户端源代码
*   `backend/`: Node.js 服务端源代码
    *   `data/`: 存放数据库文件 (`db.json`) 和配置 (`gx.txt`)
    *   `storage/`: 存放用户上传的媒体文件 (图片, 视频, 头像等)
